# Chamddl

Claude Code AI를 활용하여 개발 중인 웹 서비스 프로젝트입니다.  
AI를 기반으로 기본 구조와 기능을 빠르게 구현하고, 이후 직접 디버깅과 세부 로직을 수정하며 완성도를 높여가는 방식으로 개발하고 있습니다.

## 🛠 Tech Stack

### Frontend
- Vite
- React
- JavaScript
- TailwindCSS

### Backend
- Java
- Spring Boot

### Database
- MySQL

### DevOps
- Docker
- GitHub

## 🚀 Project Description

이 프로젝트는 Claude Code AI를 활용해 초기 코드 구조와 기능을 생성하고,  
개발자가 직접 디버깅과 리팩토링을 진행하며 웹 서비스를 구현하는 실험적인 개발 방식으로 진행되고 있습니다.

AI가 생성한 코드의 동작 원리를 분석하고 수정하면서  
다음과 같은 경험을 목표로 합니다.

- AI 기반 코드 생성 활용
- 실제 서비스 수준의 디버깅 경험
- 코드 구조 개선 및 리팩토링
- 실사용 가능한 웹 서비스 구현
